# Question Bank Sources

The platform fetches questions from these sources when you add a job.

## Implemented (fetched automatically)

| Source | Type | Content |
|--------|------|---------|
| **DataLemur** | Blog | Company SQL interview questions (Stripe, PayPal, Visa, Meta, Google, Amazon, Microsoft, Apple) |
| **StrataScratch** | Blog | Data analyst SQL/Python interview questions |
| **Interview Query** | Guides | Company interview guides (Stripe, Meta, Google, Amazon) |
| **Exercism** | API | SQLite + Python exercises (public API, no auth) |

## Referenced (for manual practice)

| Source | URL | Notes |
|--------|-----|-------|
| **Mode Analytics** | [mode.com/sql-tutorial](https://mode.com/sql-tutorial) | SQL tutorial with interactive editor |
| **LeetCode** | [leetcode.com](https://leetcode.com) | Algorithms; company tags require Premium |
| **HackerRank** | [hackerrank.com](https://hackerrank.com) | Practice; API requires enterprise |
| **CodeSignal** | [codesignal.com](https://codesignal.com) | API requires customer account |
| **Kaggle** | [kaggle.com/learn](https://kaggle.com/learn) | SQL/Python courses; may require login |
| **Project Euler** | [projecteuler.net](https://projecteuler.net) | Math/programming; has public API |
